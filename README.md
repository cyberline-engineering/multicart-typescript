# multicart-typescript

### Multicartshop API TypeScript SDK

[![Stable Release](https://img.shields.io/npm/v/@cyberline-engineering/multicartshop-client.svg)](https://npm.im/@cyberline-engineering/multicartshop-client)

Library to provide [Multicartshop&copy;](https://www.multicartshop.com) API protocol support for client-side browser-based or server-side nodejs-base JavaScript client applications.

## Table of Contents

-   [Documentation](https://dev.multicartshop.com)
-   [Installation](#installation)
-   [Building the Source](#building-the-source)

## Installation

Using [npm](https://npmjs.org/)

```sh
$ npm install @cyberline-engineering/multicartshop-client --save
```

## Building the Source

```sh
$ git clone https://github.com/@cyberline-engineering/multicartshop-client.git
$ cd cle-oidc-client
$ npm install
$ npm run build
```

### Running the Tests

```sh
$ npm test
```
